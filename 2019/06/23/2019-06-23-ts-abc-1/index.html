类型注解（Type annotations）

加上注解（其实，不加类型注解，变量类型应该是 any ）后，编译会经过类型检查，不正确会报错，但是仍然会生成相应的 js 文件。

接口（Interfaces）

描述对象字段类型。只要对象包含接口中的必须字段即可，多余字段不做检查。

类（Classes）
